What We're Building

A web-based suite of professional civil excavation tools hosted on GitHub Pages (free, static — no backend required). Organized into two tabs: Field Tools for superintendents and foremen on-site, and Project Management tools for estimators, PMs, and owners.

Live Hub URL
theexcavationacademy.github.io/tea-tools/tea-tools-hub.html
GitHub Repository
github.com/theexcavationacademy/tea-tools

Core promise: Every tool has light/dark mode, auto-save, a share link you can email, and PDF export. No login required. Works on phone or desktop. Claude handles all code and GitHub pushes — you never need to touch the repo.

Featured Tool Detail

Field Invoice Tracker

Built specifically for tracking materials, deliveries, and expenses in the field. Every entry captures:

  • Date, job name, job #, PO number
  • Invoice / delivery ticket number
  • Supplier / vendor name
  • 22 material types in dropdown (Granular A/B, clear stone, asphalt, pipe, fuel, etc.)
  • Quantity, unit, unit price — total auto-calculates
  • Dispatching contractor (who ordered) — separate from delivering trucker
  • Truck number, driver name, received by
  • Payment toggle: Company Card / Personal Reimbursement / PO / Cash
  • Multiple photo attachments (camera or upload) stored in IndexedDB
  • CSV export for accounting

PM Tools — HeavyBid / Trimble Level

  • Unit Price Builder: 12 pre-loaded labour roles with auto-fill rates (Superintendent $72/hr → Labourer $36/hr), 18 equipment types (45T Excavator $240/hr, etc.), ST/OT toggle per row, live subtotals per section, 6-layer markup stack
  • Bid Summary: Full Schedule of Values with division dropdowns (15 options), section dividers, 7-layer markup (mob → escalation → OH → profit → contingency → bond → insurance) all updating live as you type, printable bid sheet with signature blocks
  • Change Order: Force account style. OT multiplier toggle, equipment standby mode (auto 50% rate), owned/rented toggle, delay & impact costs section, complete professional CO document output

Spec Document Search

Upload any PDF specification — MMCD, city specs, engineer specs, plumbing code, any project spec. The document is read entirely in the browser using PDF.js — nothing is uploaded to any server. Supports a persistent library of multiple documents organized by job name. Works for any jurisdiction worldwide because the contractor uploads their own project-specific spec.

  • Keyword Search: Find any term across the spec, highlighted results with page references and surrounding context
  • AI Q&A: Ask natural language questions, answers grounded in the actual spec text using Claude API

TEA AI Assistant

A standalone construction AI chatbot. No spec document required. Ask anything about civil excavation, underground utilities, estimating, equipment, contracts, site safety, drainage, soils, quantities. Built as a PWA — users can "Add to Home Screen" on iPhone/Android and it opens fullscreen like a native app with no browser URL bar visible.

AI Cost at $25/month Pro Tier

Using Claude Haiku (~$0.002 per query). 100 subscribers doing 50 queries/month = approximately $10/month in API costs. Margins are ~98% — the API cost is essentially negligible at any reasonable subscription price.

AI Feature Gate — Roadmap

  • Phase 1 (Now): Users bring their own Anthropic API key — no cost to TEA, small friction for users
  • Phase 2 (Soon): Subscriber access code — paying users get a monthly code that unlocks TEA's embedded key. Low-tech, works for small subscriber counts (<100)
  • Phase 3 (Scale): Cloudflare Worker as API proxy — holds TEA's key securely server-side, validates subscription tokens. Proper SaaS architecture, still no traditional server needed

How Updates Work

  • All code is written and pushed to GitHub by Claude — you never need to touch the repo
  • Changes go to the working branch → merged to main → live on GitHub Pages within minutes
  • Every tool is a self-contained HTML file — easy to update one without touching others
  • Tools can be updated, redesigned, or improved at any time on request
